Detailed Description

A descriptor of a matrix multiplication operation.

2D case: dst[m, n] = src[m, k] * weights[k, n] + bias[m, n]

3D case: dst[mb, m, n] = src[mb, m, k] * weights[mb, k, n] + bias[mb, m, n]
